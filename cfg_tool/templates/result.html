Remove Null (ε) Productions

Grammar before Step 1

What are null productions?

A null production is any grammar rule that produces the empty string ε. For example, A → ε is a null production. These need to be removed during grammar simplification.

To remove them safely, we first find all nullable variables — variables that can derive ε either directly or indirectly. Then, for every production that contains a nullable variable, we create new versions of that rule with the nullable variable both included and omitted.

What are unit productions?

A unit production is a rule of the form A → B, where both A and B are variables (non-terminals). These are redundant because we can always substitute the rules of B directly into A.

To remove them, we find every unit pair (A, B) — meaning A can reach B through a chain of unit productions — and replace those rules with all the non-unit productions of B.

What are useless symbols?

A symbol is useless if it can never contribute to generating a terminal string from the start symbol. This happens in two ways:

  • Non-generating: The variable can never produce a string of terminals, no matter how many steps you take.
  • Unreachable: The variable can never be reached from the start symbol, so it's effectively dead code.
