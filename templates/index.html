What is CFG Simplification?

ε

Null Productions

A null production is any rule like A → ε. We remove these by generating equivalent rules that don't rely on the empty string.

A→B

Unit Productions

A unit production is a rule where one variable maps directly to another: A → B. We replace these with the actual rules of B.

∅

Useless Symbols

A symbol is useless if it can never generate a terminal string, or if it can never be reached from the start symbol.
